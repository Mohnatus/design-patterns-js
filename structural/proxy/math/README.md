# Простая переадресация

## Команда для запуска кода

```
npm run proxy:math
```

## Описание

Демо-пример.

Класс `MathProxy` служит Заместителем для класса `RealMath`. Он не выполняет никаких специфических действий, просто переадресует запросы.
