# Структурные шаблоны проектирования

## Что абстрагируется?
Способ объединения отдельных классов и объектов в более сложные структуры.

## Зачем нужны?



## Шаблоны

* Адаптер (Adapter)
* Мост (Bridge)
* Компоновщик (Composite)
* Декоратор (Decorator)
* Фасад (Facade)
* Приспособленец (Flyweight)
* Заместитель (Proxy)
